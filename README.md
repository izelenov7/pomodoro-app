## 📋 О проекте

Веб-приложение для продуктивной работы по технике **Pomodoro**. Работает прямо в браузере, не требует установки.

### Основной функционал:

**⏱️ Таймер Pomodoro**
- Круговой прогресс-бар с оставшимся временем
- Рабочий период → короткий перерыв → длинный перерыв (после 4 циклов)
- Звуковые уведомления о завершении периода
- Индикаторы пройденных циклов (12 помидоров)

**✅ Задачи**
- Список дел на день
- Быстрое добавление задач (Ctrl+Enter)
- Отметка выполнения и удаление

**📝 Заметки**
- Заметки с заголовками
- Автосохранение при вводе

**⚙️ Настройки**
- Длительность рабочего периода и перерывов
- Количество циклов до длинного перерыва
- Включение/выключение звуков

---

## 💾 Особенности

- Все данные сохраняются в **localStorage** браузера
- Таймер работает в фоновых вкладках
- Адаптивный дизайн для мобильных устройств
- Не требует регистрации
