# Pomodoro Timer — Приложение для продуктивной работы

Веб-приложение для работы по технике Pomodoro с задачами и заметками.

## 🚀 Быстрый старт

```bash
cd pomodoro-app
npm install
npm run dev  # http://localhost:5174
```

## 📦 Сборка для продакшена

```bash
npm run build  # production-сборка в dist/
npm run preview  # предпросмотр production-сборки
```

## 📁 Структура проекта

```
src/
├── components/      # UI-компоненты (Button, Input, Modal, Layout)
├── contexts/        # AppContext (глобальное состояние + логика таймера)
├── features/        # timer/, tasks/, notes/
├── hooks/           # useTimer, useLocalStorage, useNotification
├── types/           # TypeScript типы
└── utils/           # formatTime, getPhaseName, getPhaseColor
```

## ⚙️ Технологии

- **React 19** + **TypeScript**
- **Vite** — сборка и dev-сервер
- **Context API** + **useReducer** — глобальное состояние
- **localStorage** — персистентность данных
- **Web Audio API** — генерация звуков
- **Service Worker** — фоновые уведомления

## 🎯 Как это работает

### Таймер Pomodoro

1. **Рабочий период** (по умолчанию 25 минут)
   - Запустите таймер кнопкой «Старт»
   - Круговой прогресс-бар показывает оставшееся время
   - Цвет индикатора: красный

2. **Перерывы**
   - После завершения работы автоматически запускается короткий перерыв (5 мин)
   - После 4 рабочих циклов — длинный перерыв (15 мин)
   - Цвета перерывов: зелёный (короткий), синий (длинный)

3. **Индикаторы циклов**
   - Под таймером отображаются 12 шариков (3 группы по 4)
   - Закрашенный шарик = завершённый рабочий цикл
   - Разделители между группами = места для длинных перерывов
   - После 12 помидоров индикаторы автоматически сбрасываются

4. **Управление**
   - **Старт/Пауза** — запуск и остановка таймера
   - **Сброс** — возврат к началу текущего периода
   - **Пропустить** — досрочное завершение перерыва
   - **Сброс счётчика** — обнуление количества помидоров

### Задачи

- Добавление задач через многострочный ввод
- **Ctrl+Enter** — быстрое добавление задачи
- Чекбоксы для отметки выполнения
- Удаление завершённых задач

### Заметки

- Заметки с заголовками
- Автосохранение при вводе
- Поддержка форматирования текста

### Настройки

Нажмите на иконку ⚙️ в таймере для настройки:

- Длительность рабочего периода (мин)
- Длительность короткого перерыва (мин)
- Длительность длинного перерыва (мин)
- Количество циклов до длинного перерыва
- Звуковые уведомления (вкл/выкл + выбор звука)

## 💾 Хранение данных

Все данные сохраняются в **localStorage** браузера:

- `pomodoro_settings` — настройки таймера
- `pomodoro_timer_state` — текущее состояние таймера
- `pomodoro_tasks` — список задач
- `pomodoro_notes` — текст заметок

Данные сохраняются автоматически и восстанавливаются при перезагрузке страницы.

## 🔔 Уведомления

- **Звуковые** — Web Audio API генерирует звук без внешних файлов
- **Браузерные** — Service Worker показывает уведомления даже в фоновых вкладках

## 🎨 Особенности

- Таймер продолжает работать при переключении между вкладками
- Компенсация задержек при потере фокуса вкладкой
- Адаптивный дизайн для мобильных устройств
